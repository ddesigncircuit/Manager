Setup Guide

1. Go to emailjs.com and create a free account

2. Add an Email Service (Gmail recommended) → copy service_id

3. Create an Email Template with these variables:

{{to_email}}Recipient email address
{{to_name}}Recipient name
{{subject}}Email subject line
{{message}}Intro message (also included inside doc_html)
{{doc_html}}The complete email — set as your template body

4. In your EmailJS template, set the body/content field to just: {{{doc_html}}} (triple braces to allow HTML)

5. Go to Account → API Keys → copy your public_key
