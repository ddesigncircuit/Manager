Email Settings

EmailJS Configuration
Configure EmailJS to send documents directly from the app. Settings are stored locally in your browser.
Setup Guide
How to get your EmailJS credentials:
  1. Visit emailjs.com and create a free account
  2. Add an Email Service (Gmail works well) — note the Service ID
  3. Create an Email Template — note the Template ID
  4. Go to Account → API Keys for your Public Key
Template variables your EmailJS template should use:
{{to_email}}  — recipient address
{{to_name}}   — recipient name
{{subject}}   — email subject
{{message}}   — full message body
{{from_name}} — The Design Circuit
{{reply_to}}  — reply-to address
